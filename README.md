# HW2 - Simple GenAI Workflow

## Workflow Chosen
The goal of this project is to summarize meeting content into actionable items. To run the code, set your own GEMINI_API_KEY as an environment variable before executing app.py.

## Target Users
The target users are business students, analysts, or anyone who relies heavily on meetings to advance their work and learning, as well as team members who need to collaborate on assignments. These users often need to organize scattered, hastily taken meeting notes into structured follow-up items.

## Input
The system accepts:
- raw meeting transcripts  
- bullet-point notes uploaded by users  
- memo-style meeting records  

## Output
The system will generate:
- a brief meeting summary  
- a set of action items  

If explicitly mentioned in the text, it will also include necessary project details such as the person in charge, deadlines, and specific requirements.

## Why This Task Matters
Meetings often result in disorganized notes. The thought process at the time of note-taking may not align with the perspective when reviewing them later, which can lead to inconsistencies. In longer meetings, people may lose focus, miss key points, or misunderstand certain details.

Organizing these notes usually requires a lot of repetitive work. By introducing AI, this process can become more consistent and efficient, helping teams focus more on execution rather than spending time verifying and reorganizing meeting content.
